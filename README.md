# angular-phonecat-iscroll-demo


## Overview
Angular Phonecat Demo porting for angular-iscroll (https://github.com/ibspoof/ng-iScroll). For study purpose.

### Karma
- http://karma-runner.github.io/

- http://pivotal.github.io/jasmine/
